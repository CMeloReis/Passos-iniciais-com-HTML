Responsividade

O layout responde ao tamanho da tela.

Layout responsivo: Página que se adapta a qualquer tamanho de dispositivo. Ou seja, os elementos serão organizados do menor ao maior tamanho.

Mobile First

Conceito de Mobile First: Fazer o desenvolvimento inicialmente pensando em telas de celular, para depois elas serem pensadas para telas de desktop. Contudo, esse conceito evoluiu para um entendimento em que deve-se desenhar o site pensando na maior tela que irá abri-lo, pensando também como irá se comportar na menor tela que o site irá aparecer.
